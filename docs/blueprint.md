# **App Name**: Portugal Pathway

## Core Features:

- Landing Page: A hero section introduces the system with an engaging visual layout and a call-to-action button for users to sign up or log in.
- User Authentication: A visual login form with error, success, and loading states enhances the user experience. Allows standard username/password entry.
- Dashboard Overview: An informative dashboard presents progress, document count, and total costs using cards, progress bars, badges, and icons.
- User Management: Allows for admins to view a list of current users and add a new user.
- Task Management: Tasks are listed in cards or a table with visual status indicators and filters to organize pending, in-progress, and completed items, enhancing task workflow.
- Document Handling: Supports drag & drop uploads of documents and also viewing of current documents.
- AI-Powered Chat Assistant: AI tool assesses user inputs within the chat interface to give information or redirect user towards an existing tool for specific topics, simplifying immigration planning and offering personalized guidance.

## Style Guidelines:

- Choose a light color scheme to maintain focus. Background color: Light Gray (#f0f0f0) for a clean backdrop.
- Primary color: Blue (#3498db) for trust and clarity.
- Accent color: Orange (#e67e22) to highlight calls to action and important information, providing contrast and guiding user attention.
- Headline font: 'Poppins' (sans-serif) for a clean, modern feel, suitable for titles and headings.
- Body font: 'PT Sans' (sans-serif) provides readability and complements the clean interface design.
- Use minimalist icons from 'Line Awesome' for a modern and clean look. Use icons consistently across the interface to represent actions and categories.
- Employ a grid-based layout for consistent alignment and spacing. Use 'Tailwind CSS' utility classes for responsive adjustments.
- Apply subtle transitions and animations to loading states, form submissions, and navigation to provide feedback and improve user experience.
